## Animation Sequences

### Opening Transaction Details
```
1. User taps transaction tile
2. Tap feedback (ripple animation)
3. Overlay fade-in (0.3s)
4. Bottom sheet slide-up (0.4s easing curve)
5. Content appears (fade-in 0.2s)
```

### Applying Filter
```
1. User taps filter chip
2. Chip selection animation
3. Picker dialog appears (fade & scale)
4. User selects option
5. List cross-fade animation
6. New filtered content appears (0.3s)
```

### Adding Transaction
```
1. User taps FAB
2. FAB rotation animation (90Â°)
3. Screen transition (slide from right)
4. Returns to history
5. New transaction slides in at top
```
